# Praktikum Dasar Kecerdasan Artifisial

## Deskripsi

Praktikum ini bertujuan untuk memberikan pemahaman dasar tentang konsep dan implementasi kecerdasan artifisial (AI).
Peserta akan mempelajari berbagai algoritma dan teknik yang digunakan dalam AI, serta bagaimana mengaplikasikannya dalam
berbagai kasus nyata.
